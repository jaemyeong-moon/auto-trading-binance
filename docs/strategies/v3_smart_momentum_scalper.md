## 한줄 요약

실전과 백테스트의 괴리를 줄이기 위해, ATR 기반 동적 TP/SL + 실행비용 사전 차감 + 리스크-리워드 2:1 보장 + 과다매매 방지를 적용한 전략.

---

## 왜 v3가 필요한가 (v1/v2의 실전 문제)

| 문제 | 원인 | v3 해결 |
|------|------|---------|
| 백테스트 > 실전 | 슬리피지/스프레드 미반영 | 실행비용 0.06% 사전 차감 후 RR 계산 |
| 변동성 무시 TP/SL | 고정 1%/0.5% | ATR × 배수로 동적 조절 |
| 과다매매 | 매 틱마다 진입 시도 | 시간당 4회 제한 |
| 가짜 추세 진입 | EMA만으로 판단 | ADX > 30 + DI 방향 확인 |
| 불리한 가격 진입 | 가격 위치 무시 | VWAP 아래서 롱, 위에서 숏 |
| 후행 지표 | EMA/RSI 후행 | 모멘텀 '가속도' 확인 |

## 전략 구조

### 1. 시장 체제 판단 (ADX 중심)

```
ADX > 30           → 강한 추세 (매매 허용)
ADX 20~30          → 약한 추세 (매매 허용)
ADX < 20           → 횡보 (매매 금지)
ATR > 평균 × 2.0   → 극변동성 (매매 금지)
```

**v2와의 차이:** EMA 정렬이 아닌 ADX로 추세 '강도'를 수치로 측정. 방향은 DI+/DI-로 판단.

### 2. 진입 점수 시스템 (7점 만점, 5점 이상 진입)

| # | 지표 | +1 조건 | 의미 |
|---|------|---------|------|
| 1 | ADX 방향 | DI+>DI- (LONG) | 추세 방향 확인 |
| 2 | 15분봉 동의 | HTF EMA8>EMA21 | 상위 프레임 추세 일치 |
| 3 | VWAP 위치 | 가격 < VWAP (LONG) | 기관 가격 대비 유리 |
| 4 | 거래량 폭발 | 현재 > 평균 × 1.5 | 강한 참여 (v2의 1.2보다 엄격) |
| 5 | 모멘텀 가속 | ROC 변화율 양수 | 움직임이 가속 중 (후행성 보완) |
| 6 | RSI 영역 | 35~65 | 과매수/과매도 아님 |
| 7 | MACD 강화 | 히스토그램 커지는 중 | 단순 양수가 아닌 '성장' |

### 3. 리스크-리워드 사전 검증

```
예상 TP = ATR × 2.0
예상 SL = ATR × 1.0

실질 TP = 예상 TP - (가격 × 0.06%)   ← 수수료+슬리피지 차감
실질 SL = 예상 SL + (가격 × 0.06%)   ← 손절은 비용만큼 더 손해

실질 RR = 실질 TP / 실질 SL
RR < 2.0 → 진입 안함
```

**왜 중요한가:** 백테스트에서 수익인 매매가 실전에서 수수료 때문에 손실이 되는 걸 방지.

### 4. ATR 기반 동적 TP/SL

| 항목 | 값 | 예시 (ATR=$500) | 예시 (ATR=$50) |
|------|----|-----------------|----------------|
| 손절 | 1.0 × ATR | $500 (0.7%) | $50 (0.07%) |
| 부분익절 | 1.0 × ATR | $500에서 50% 청산 | $50에서 50% 청산 |
| 전량익절 | 2.0 × ATR | $1,000 | $100 |
| 트레일링 활성화 | 1.5 × ATR | $750 수익 시 | $75 수익 시 |
| 트레일링 거리 | 0.5 × ATR | 고점에서 $250 | 고점에서 $25 |

**핵심:** 시장이 크게 움직이면 TP/SL도 넓히고, 잔잔하면 좁힌다. 고정 %가 아닌 시장 호흡에 맞춤.

### 5. 과다매매 방지

- **시간당 최대 4회** 매매 (시간 바뀌면 리셋)
- **ATR이 수수료의 3배 미만이면 진입 안함** (움직임이 너무 작으면 수수료만 나감)

### 6. 점진적 쿨다운

```
1회 손절 → 5봉 쿨다운 (약 5분)
2연패   → 9봉 (약 9분)
3연패   → 11봉 (약 11분)
...
최대    → 20봉 (약 20분)

수익 1회 → 즉시 리셋
```

**v2와의 차이:** 고정 4봉이 아닌, 연패가 심할수록 더 오래 대기. 시장이 본인에게 불리할 때 빠르게 인정하고 쉬는 것.

## 엔진 동작 흐름

```
매 틱 (30초)
  │
  ├─ 1분봉 300개 + 15분봉 100개 조회
  │
  ├─ 쿨다운 중? → 패스
  ├─ 시간당 매매 제한 초과? → 패스
  ├─ ATR < 수수료 × 3? → 패스 (수익 불가능)
  │
  ├─ 15분봉 → ADX 체제 판단
  │   ├─ 횡보/극변동성 → 매매 금지
  │   └─ 추세 → 계속
  │
  ├─ 포지션 있으면
  │   ├─ ATR 기반 동적 SL 도달? → 청산 + 쿨다운
  │   ├─ 부분 TP 도달? → 50% 청산
  │   ├─ 트레일링 활성화? → 트레일링 스탑 설정
  │   ├─ 체제 변경(→횡보)? → 청산
  │   └─ 유지
  │
  ├─ 포지션 없으면
  │   ├─ 7개 지표 점수 계산
  │   ├─ 5점 미만? → 대기
  │   ├─ RR < 2.0? → 대기
  │   └─ 진입
  │
  └─ 손익 기록 → 점진적 쿨다운 업데이트
```

| 항목 | v1 | v2 | v3 |
|------|----|----|------|
| TP/SL | 고정 % | 고정 % | **ATR 동적** |
| 수수료 | 무시 | 무시 | **사전 차감** |
| 추세 판단 | EMA 크로스 | EMA+BB+ATR | **ADX 강도** |

| 매매 빈도 | 무제한 | 무제한 | **시간당 4회** |
| 가격 앵커 | 없음 | 없음 | **VWAP** |
| 모멘텀 | 방향만 | 방향만 | **가속도** |

## 알려진 한계

1. **ADX 후행성** — ADX 자체도 14봉 기반이라 추세 초기를 놓침
2. **VWAP 리셋** — 일봉 기준 리셋이 아닌 캔들 누적이라 장시간 운용 시 왜곡 가능
3. **시간당 4회 제한** — 강한 추세에서 기회를 놓칠 수 있음
4. **ATR 배수 고정** — 1.0/2.0 배수가 최적인지 시장별 검증 필요
