## 한줄 요약

모멘텀 폭발을 감지하면 즉시 진입, 타이트한 TP/SL로 빠르게 먹고 빠진다. 횡보에서도 매매하며, 쿨다운 없이 연속 거래한다.

---

## v1~v3와 근본적 차이

| 항목 | v1~v3 | v4 |
|------|-------|------|
| 철학 | 확실할 때만 진입 | **망설이면 늦다, 일단 진입** |
| 진입 기준 | 3~5/6~7점 | **2/4점 (낮은 장벽)** |
| 횡보 처리 | 매매 금지 | **스퀴즈 돌파 + RSI 역추세** |
| 쿨다운 | 4~20봉 | **없음 (즉시 재진입)** |
| TP/SL | 넓음 (1~2 ATR) | **타이트 (0.5/0.75 ATR)** |
| 부분익절 | 1 ATR | **0.3 ATR (초빠른)** |
| 연패 처리 | 쿨다운 증가 | **방향 반전 (flip mode)** |
| 거래 빈도 | 낮음 | **높음** |

## 전략 구조

### 1. 마이크로 시장 상태 (초단기)

| 상태 | 감지 조건 | 행동 |
|------|-----------|------|
| **모멘텀 UP** | 3봉간 > 0.8 ATR 상승 | LONG 진입 |
| **모멘텀 DOWN** | 3봉간 > 0.8 ATR 하락 | SHORT 진입 |
| **스퀴즈** | BB폭 < 평균의 60% | 돌파 방향 대기 → 진입 |
| **무질서** | 위 해당 없음 | RSI 극단에서 역추세 |

### 2. 진입 점수 (4점 만점, 2점이면 진입)

| # | 지표 | 기준 | 특징 |
|---|------|------|------|
| 1 | 초고속 EMA(3/5) | 크로스 방향 일치 | 후행 최소화 |
| 2 | 거래량 스파이크 | > 평균 2배 | 강한 참여 |
| 3 | 대형 캔들 | 몸통 > 70% | 확신 캔들 |
| 4 | 15분봉 동의 | HTF EMA 방향 | 보너스 |

**2점이면 진입** — v3의 5/7점 대비 압도적으로 낮은 장벽.

### 3. 방향 결정 (시장 상태별)

```
모멘텀 → 폭발 방향 그대로 (순추세)
스퀴즈 → 대형 캔들 or EMA 방향 (돌파 기대)
무질서 → RSI < 25면 LONG, RSI > 75면 SHORT (평균회귀)
```

### 4. TP/SL (타이트)

| 항목 | ATR 배수 | BTC 예시 (ATR=$300) |
|------|----------|-------------------|
| 손절 | **0.5×** | $150 |
| 부분익절 (50%) | **0.3×** | $90에서 절반 확보 |
| 전량익절 | **0.75×** | $225 |

**v3 대비 3배 좁다.** 빠르게 먹고 빠짐 → 보유 시간 짧음 → 변동성 노출 최소화.

### 5. 연패 반전 (Flip Mode)

```
2연패 → flip mode ON (신호 방향 반전)
다시 2연패 → flip mode OFF (원래대로)
1회 수익 → flip mode OFF + 리셋
```

**왜:** 연패 = 시장 방향을 잘못 읽고 있다는 의미. 뒤집으면 맞는 방향으로 전환.
쿨다운(쉬기)보다 적극적인 대응.

### 6. 모멘텀 반전 즉시 청산

포지션 보유 중 반대 방향으로 1.0 ATR 이상 모멘텀 폭발이 감지되면 → 즉시 CLOSE.
TP/SL 도달 전이라도 모멘텀이 뒤집히면 바로 나감.

## 엔진 동작 흐름

```
매 틱 (30초)
  │
  ├─ 쿨다운? → 없음 (항상 분석)
  │
  ├─ 포지션 있으면
  │   ├─ 반대 모멘텀 폭발? → 즉시 청산
  │   ├─ SL 0.5 ATR 도달? → 청산
  │   ├─ 부분 TP 0.3 ATR? → 50% 청산
  │   ├─ TP 0.75 ATR? → 전량 청산
  │   └─ 유지
  │
  ├─ 포지션 없으면
  │   ├─ 마이크로 상태 판단
  │   ├─ 방향 결정
  │   ├─ flip mode? → 방향 반전
  │   ├─ 점수 >= 2? → 진입
  │   └─ 점수 < 2 → 다음 틱
  │
  └─ 손익 기록 → flip mode 업데이트
```

## 리스크

이 전략은 의도적으로 공격적입니다:

1. **높은 거래 빈도 → 수수료 누적** — 수익이 수수료를 커버해야 함
2. **타이트 SL → 더 자주 손절** — 승률이 50% 이상이어야 수익
3. **횡보 매매 → 양쪽으로 손절 가능** — RSI 역추세가 안 먹힐 때
4. **flip mode → 연속 반전은 최악** — 시장이 무질서하면 방향 반전도 안 먹힘

**레버리지와 결합하면 수익도 크지만 손실도 빠르다.** 투자 비율 설정에 주의.
