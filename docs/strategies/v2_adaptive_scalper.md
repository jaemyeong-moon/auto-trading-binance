## 한줄 요약

시장 상태(추세/횡보/변동성)를 자동 판단하고, 횡보 시 진입을 금지하며, 점수 기반으로 신호가 강할 때만 진입한다. 손절 후 쿨다운, 부분익절 + 트레일링 스탑.

---

## v1과의 차이

| 항목 | v1 (Momentum Flip) | v2 (Adaptive) |
|------|---------------------|---------------|
| 포지션 유지 | 항상 보유 | 신호 있을 때만 |
| 시장 판단 | 없음 | ATR + BB + EMA |
| 횡보 처리 | 플립 반복 (손실) | 진입 금지 |
| 진입 기준 | EMA 크로스 1개 | 5개 지표 점수 (3/5) |
| 손절 후 | 즉시 재진입 | 쿨다운 대기 (4봉) |
| 익절 | 전량 1회 | 부분(50%) + 트레일링 |
| 역추세 모드 | 3연패 시 반전 | 없음 (대기로 해결) |

## 전략 구조

### 1. 시장 상태 판단

```
EMA(8) > EMA(21) > EMA(50) + BB폭 정상 → 상승 추세
EMA(8) < EMA(21) < EMA(50) + BB폭 정상 → 하락 추세
ATR > 평균의 1.5배 → 변동성
그 외 → 횡보
```

| 상태 | 조건 | 행동 |
|------|------|------|
| 상승 추세 | EMA 정렬 ↑ | LONG 방향 진입 판단 |
| 하락 추세 | EMA 정렬 ↓ | SHORT 방향 진입 판단 |
| 변동성 | ATR 급등 | EMA 방향만 따름 |
| 횡보 | 위 해당 없음 | **진입 금지 (No Trade Zone)** |

### 2. 진입 점수 시스템 (5점 만점)

| 지표 | +1 조건 (LONG) | +1 조건 (SHORT) |
|------|----------------|-----------------|
| EMA 정렬 | EMA(8) > EMA(21) | EMA(8) < EMA(21) |
| 거래량 | 현재 > 평균 × 1.2 | 현재 > 평균 × 1.2 |
| RSI | 30 < RSI < 70 | 30 < RSI < 70 |
| MACD | 히스토그램 > 0 | 히스토그램 < 0 |
| BB 위치 | 가격이 하단 근처 | 가격이 상단 근처 |

**3점 이상이면 진입, 미만이면 대기.**

### 3. No Trade Zone (횡보 필터)

횡보로 판단되면 어떤 점수가 나오든 진입하지 않는다.
→ v1에서 횡보장 잦은 플립으로 인한 수수료 손실 문제 해결.

### 4. 쿨다운

```
손절 발생 → 4봉(약 4분) 대기
대기 중 모든 신호 무시 (HOLD)
쿨다운 종료 후 정상 평가 재개
```

**왜 필요한가:** 손절 직후는 변동성이 높고 감정적 진입이 발생하기 쉬움.
봉 수 기준으로 대기하면 시장이 안정될 시간을 확보.

### 5. 포지션 관리

**항상 보유 X** — 강한 신호가 있을 때만 진입.

포지션 청산 조건:
- 추세 역전 감지 시 → CLOSE 신호
- 트레일링 스탑 발동 시 → CLOSE 신호
- 엔진의 TP/SL → 강제 청산

### 6. 익절 구조

```
가격 +0.5% → 포지션 50% 부분 익절
가격 +0.8% → 트레일링 스탑 활성화 (거리 0.3%)
가격 +1.2% → 전량 익절
```

**트레일링 동작 (LONG 예시):**
1. 진입가 $100, 가격이 $100.80 도달 → 트레일링 활성화
2. 가격이 $101.50까지 상승 → 트레일링 스탑 = $101.50 × 0.997 = $101.20
3. 가격이 $101.20 이하로 하락 → 청산

**부분 익절 의의:**
- 수익 일부를 확보하여 심리적 안정
- 나머지 50%는 트레일링으로 추가 수익 추구

### 7. 손절

| 항목 | 값 |
|------|------|
| 손절 | 0.5% (엔진 설정값 사용) |
| 손절 후 | 쿨다운 4봉 → 재진입 없음 |

## 알려진 한계

1. **추세 초기 진입 느림** — EMA(50)까지 정렬돼야 추세로 판단하므로, 빠른 추세 전환 초기를 놓칠 수 있음
2. **쿨다운이 고정** — 시장 상황과 무관하게 4봉 대기. 변동성에 따라 동적 조절 필요
3. **부분익절 후 잔여 포지션 관리** — 거래소에서 부분 청산이 실패하면 상태 불일치 가능
4. **점수 가중치 없음** — 5개 지표에 동일 가중치. 시장에 따라 특정 지표가 더 중요할 수 있음
5. **시장 상태 전환 지연** — 횡보 → 추세 전환 시 EMA가 아직 정렬 안 돼서 첫 움직임을 놓침

- [ ] 진입 점수 임계값을 대시보드에서 조절 가능하게
- [ ] ADX 지표 추가로 추세 강도 더 정밀하게 판단
- [ ] 복수 타임프레임 (1분 + 5분) 동시 참조
